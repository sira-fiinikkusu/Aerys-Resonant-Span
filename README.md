# Aerys

A contextually-aware personal AI assistant built on n8n, featuring multi-platform messaging (Discord + Telegram), persistent memory with pgvector, per-tier model routing (Gemini/Sonnet/Opus via OpenRouter), identity resolution across platforms, and extensible sub-agents for research, email, and media analysis.
