Create the 05-02 research sub-agent workflow in n8n via the API. Structure:

**Entry:**
1. Execute Workflow Trigger (typeVersion 1, passthrough)
2. Read Input (Code node):
   ```javascript
   const input = $('Execute Workflow Trigger').first().json;
   return [{json: {
     query: input.query,
     person_id: input.person_id,
     original_message: input.original_message || input.query,
     source_channel: input.source_channel,
     conversation_privacy: input.conversation_privacy
   }}];
   ```

**Search:**
3. Tavily Search (HTTP Request node):
   - Method: POST
   - URL: https://api.tavily.com/search
   - Authentication: Header Auth (or generic Header field with $vars.TAVILY_API_KEY)
   - Body (JSON):
     ```json
     {
       "query": "={{$json.query}}",
       "search_depth": "advanced",
       "include_answer": "advanced",
       "max_results": 5,
       "include_raw_content": false,
       "topic": "general"
     }
     ```
   - Error handling: onError continueRegularOutput (so LLM step can handle search failures)

**Format for synthesis:**
4. Format Research Context (Code node):
   ```javascript
   const tavily = $input.first().json;
   const input = $('Read Input').item.json;

   // Handle Tavily error case
   if (tavily.error || !tavily.answer) {
     return [{json: {
       research_failed: true,
       query: input.query,
       error_message: tavily.error || 'No results returned'
     }}];
   }

   const answer = tavily.answer || '';
   const sources = (tavily.results || [])
     .slice(0, 3)
     .map(r => `- [${r.title}](${r.url})`)
     .join('\n');

   return [{json: {
     research_context: `## Web Research\n\n${answer}\n\nSources:\n${sources}`,
     query: input.query,
     raw_answer: answer,
     sources_text: sources,
     research_failed: false
   }}];
   ```

**LLM synthesis:**
5. Synthesize in Aerys Voice (HTTP Request POST to OpenRouter):
   - Condition: only runs if research_failed === false (add IF node before this step)
   - model: anthropic/claude-haiku-4.5
   - System: "You are Aerys. You have just performed a web search on behalf of the user. Present the findings naturally in your own voice — curious, engaged, direct. Start with a subtle acknowledgment that you looked this up (e.g. 'I looked into this...' or 'I searched around and found...'). Synthesize the key information clearly. End with 3 sources in a short list under 'Sources:'. Do NOT just repeat the raw search answer verbatim — add your perspective and framing."
   - User: `User asked: {{$('Read Input').item.json.original_message}}\n\nSearch results:\n{{$json.research_context}}`
   - max_tokens: 800

6. Parse Synthesis (Code node):
   ```javascript
   const synthesis = $input.first().json;
   const query = $('Read Input').item.json.query;

   // Handle synthesis error or fallback path
   if ($json.research_failed) {
     return [{json: {
       result: `I tried to look that up but ran into a search hiccup. ${$json.error_message || 'The search returned nothing usable.'} I can try again or answer from what I already know — your call.`,
       query
     }}];
   }

   const text = synthesis?.choices?.[0]?.message?.content || $('Format Research Context').item.json.research_context;
   return [{json: {result: text, query}}];
   ```

**Return:**
7. Return Result (Code node): `return [{json: {result: $json.result, query: $json.query}}]`
